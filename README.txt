Instruccions per fer funcionar el programa:


-Descarregar MAMP per fer la base de dades
-Obrir MAMP, clicar a Preferences Apache port 8080, MySQL port 3307, despres OK. ASI Inicar Servidores. Sobrirà el navegador amb la pàgina de MAMP.Anar a 
Tools->phpMyAdmin. Clicar a Nou. A Crear base de dades, ficar de nom dpo_troner i crear-la. 
A partir de la nova base de dades, clicar a Importa i seleccionar larxiu de la base de dades.
-Obrir IntelliJ. Els servidors del MAMP han destar iniciats. Compilar primer el Troner_Servidor i despres el Troner_Client
